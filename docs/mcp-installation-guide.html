⚠️ IMPORTANT: You need Claude Desktop (downloaded app) to use these writing tools. The web version at claude.ai cannot connect to local MCPs. This tutorial will take 1-2 hours to complete, with most time spent downloading software.

📋 System Requirements

💻 Windows Requirements

  • Windows 10 or newer
  • 4GB RAM minimum
  • 2GB free disk space
  • Administrator access

🍎 Mac Requirements

  • macOS 10.15 or newer
  • 4GB RAM minimum
  • 2GB free disk space
  • Admin password access

📦 Software Installation Steps

1. Claude Desktop 🤖

Why we need this: Claude Desktop can connect to your local writing tools. The web version cannot.

  1. Go to https://claude.ai/download
  2. Click "Claude for Windows" - downloads a .exe file
  3. Run the downloaded .exe file
  4. Follow the installation prompts (accept defaults)
  5. Open Claude Desktop and sign in with your Claude account
📸 Visual Check: You should see a desktop app window, NOT a web browser
  1. Go to https://claude.ai/download
  2. Choose your Mac type:
    • Apple Silicon (M1/M2/M3): "Claude for macOS (Apple Silicon)"
    • Intel Mac: "Claude for macOS (Intel)"
    • Not sure? Apple menu → "About This Mac" → Look at "Chip"
  3. Open the downloaded .dmg file
  4. Drag Claude to your Applications folder
  5. Open Claude from Applications and sign in
📸 Visual Check: Claude appears in your Applications folder and opens as a desktop app
Download Claude Desktop

2. Visual Studio Code 📝

Why we need this: VS Code is where you'll view and modify the MCP code with AI assistance.

  1. Go to https://code.visualstudio.com/download
  2. The website should auto-detect your operating system
  3. Click the large download button
  4. Windows: Run the installer, accept defaults
  5. Mac: Open the .zip file, drag VS Code to Applications
  6. Open VS Code - you should see a welcome screen
📦 Install Essential Extensions:
  1. Click the Extensions icon (four squares) in the left sidebar
  2. Search for and install these one by one:
    • "GitHub Copilot" - AI coding assistant
    • "GitHub Copilot Chat" - Chat with AI
    • "Docker" - Container management
    • "PostgreSQL" - Database tools
    • "PowerShell" - Windows terminal support
Download VS Code

3. GitHub Account Setup 🐙

Why we need this: GitHub Copilot will help you modify code, and we'll download our MCP code from GitHub.

  1. Go to https://github.com
  2. Sign up for a free account if you don't have one
  3. In VS Code, press Ctrl+Shift+P (Windows) or Cmd+Shift+P (Mac)
  4. Type "GitHub Copilot: Sign In" and follow prompts
💡 Note: You don't need to pay for Copilot subscription - we'll use API keys for assistance.
Create GitHub Account

4. Docker Desktop 🐳

Why we need this: Docker runs your PostgreSQL database easily without complex database installation.

  1. Go to https://docs.docker.com/desktop
  2. Click "Docker Desktop for Windows"
  3. Run the installer
  4. Important: Restart your computer when prompted
  5. Open Docker Desktop after restart
  6. Accept the license agreement
  7. Look for the Docker whale icon in your system tray
🔧 Common Windows Issues:
  • Virtualization: May need to enable in BIOS
  • WSL 2: Enable if prompted by Docker
  • Won't start: Restart computer and try again
  1. Go to https://docs.docker.com/desktop
  2. Choose your Mac type:
    • Intel Mac: "Docker Desktop for Mac with Intel chip"
    • Apple Silicon: "Docker Desktop for Mac with Apple chip"
  3. Open the downloaded .dmg file
  4. Drag Docker to Applications
  5. Open Docker Desktop from Applications
  6. Grant permissions when asked
  7. Look for the Docker whale icon in your menu bar
📸 Success Check: Docker whale icon visible in system tray/menu bar
Download Docker Desktop

5. Node.js ⚡

Why we need this: Node.js runs the MCP server code that connects your database to Claude.

  1. Go to https://nodejs.org
  2. Download the LTS version (Long Term Support - more stable)
  3. Should be version 18 or higher
  4. Windows: Run the .msi installer with default settings
  5. Mac: Run the .pkg installer with default settings
✅ Verify Installation:
  1. Windows: Open Command Prompt (search "cmd")
  2. Mac: Open Terminal
  3. Type: node --version
  4. You should see something like: v18.17.0 or higher
Download Node.js

6. Git (Windows Only) 📦

Why we need this: Git downloads code repositories from GitHub.

🍎 Mac Users: Git is already installed on Mac! Skip this step.

Windows Installation:

  1. Go to https://git-scm.com/download/win
  2. Download and run the installer
  3. Important options:
    • Choose "Git from the command line and also from 3rd-party software"
    • Choose "Use bundled OpenSSH"
    • Use default settings for everything else
Download Git (Windows)

🔍 Verification Checklist

Test Everything is Working:

Open Command Prompt (Windows) or Terminal (Mac) and run these commands:
  • Node.js: node --version → Should show v18+ or higher
  • Git: git --version → Should show version information
  • Docker: Look for whale icon in system tray/menu bar
  • VS Code: Opens and shows GitHub Copilot extensions installed
  • Claude Desktop: Opens as desktop app (not web browser)

🔧 Troubleshooting Common Issues

Claude Desktop Won't Open

  • Make sure you downloaded Desktop version, not using web browser
  • Check system requirements on download page
  • Try restarting your computer

Docker Won't Start

  • Windows: Enable virtualization in BIOS, install WSL 2 if prompted
  • Mac: Check System Preferences → Security & Privacy for blocked software
  • All: Restart computer and try again

"Command Not Found" Errors

  • Restart Command Prompt/Terminal after installing software
  • Windows: Make sure software was added to PATH during installation
  • Try opening a new terminal window
